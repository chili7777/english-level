# 📚 English Learning Reference Guide
## 🚀 CÓMO ABRIR LA APLICACIÓN
### Opción 1: Usar el archivo START_SERVER.bat (MÁS FÁCIL)
1. Haz doble clic en **START_SERVER.bat**
2. Se abrirá automáticamente tu navegador en http://localhost:8000
3. ¡Listo! Ya puedes usar la aplicación
### Opción 2: Manualmente con PowerShell
1. Abre PowerShell en esta carpeta
2. Ejecuta: ```python -m http.server 8000```
3. Abre tu navegador y ve a: http://localhost:8000
### Opción 3: Con VS Code
1. Instala la extensión "Live Server"
2. Click derecho en index.html
3. Selecciona "Open with Live Server"

## ❓ Problemas Comunes
**Error: "Error loading data"**
- Solución: Usa el archivo START_SERVER.bat
- NO abras index.html directamente con doble clic
- DEBES usar un servidor local (http://)
**Puerto 8000 ya está en uso**
- Cierra otras instancias del servidor
- O usa otro puerto: ```python -m http.server 8080```
